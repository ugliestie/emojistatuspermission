# Telegram Web App for Emoji Status permission

Example HTML-file that contains a plain-JS interaction with Telegram Web Apps API for setting emoji status.

## Links
* Official docs: https://core.telegram.org/bots/webapps
* Repo where I found the base for it: https://github.com/revenkroz/telegram-web-app-bot-example

## How to use it?
It's simple: just link this site to button and it will open blank page with one button for permission by user! When user agree, it closes automatically
